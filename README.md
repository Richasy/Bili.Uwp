`哔哩` 是一款 [哔哩哔哩](https://www.bilibili.com) 的第三方应用，使用 UWP 框架开发，是原生的 Windows 应用，支持 Windows 10/11 系统。以设计和易用性为主，广受用户好评。

## 🙌 简单的开始

如果想安装哔哩，请打开右侧的 [Release](https://github.com/Richasy/Bili.Uwp/releases) 页面，找到最新版本，并选择适用于当前系统的安装包下载。

然后打开系统设置，进入 `隐私与安全` ，选择 `开发者选项` ，打开 `开发者模式` ，并等待系统安装一些必要的扩展项。

在应用压缩包下载完成后，解压压缩包，并在管理员模式下，使用 PowerShell 运行解压后的 `install.ps1` 脚本，根据提示进行安装。

关于如何一步步地使用侧加载 (sideload) 方式安装 UWP 应用，请参见 [下载并安装哔哩的详细说明](./how_to_install.md) 。
